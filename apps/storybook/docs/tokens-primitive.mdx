# Primitive Tokens

- 머먹지 디자인 시스템의 기초 디자인 토큰입니다.
- CSS 변수로 제공되며, **색상/타이포그래피/간격** 등 UI의 기본 단위를 정의합니다.

## 설치 Install

```zsh
# npm
npm install @mumukji/tokens

# pnpm
pnpm add @mumukji/tokens
```

## 사용법 Usage

CSS 또는 TS/JS 파일에서 import하면 `:root`에 CSS 변수와 A2z 웹폰트가 자동으로 등록됩니다.

```ts
import '@mumukji/tokens/css';
```
