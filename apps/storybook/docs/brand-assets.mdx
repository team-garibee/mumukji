# Brand Assets

- 머먹지 서비스의 브랜딩 에셋을 모아둔 비공개 패키지입니다.
- npm에 배포하지 않으며, 모노레포 내부(`workspace:*`)에서만 사용합니다.

<br />

## Usage

```zsh
pnpm add @mumukji/brand-assets
```

```tsx
import { Wordmark, mongjiDefault, mukjiDancing } from '@mumukji/brand-assets';
```

## Lotties

Lottie JSON 데이터만 제공합니다. 재생은 소비 측에서 원하는 라이브러리를 사용하세요.

```tsx
import Lottie from 'lottie-react';
import { mukjiDancing } from '@mumukji/brand-assets';

<Lottie animationData={mukjiDancing} loop />;
```
